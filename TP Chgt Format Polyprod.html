📋 Mise en situation

Vous êtes conducteur de ligne de conditionnement dans un atelier utilisant la machine Polyprod PP30 (ERM Automatismes Industriels) intégrée à la ligne ERMAFLEX. Aujourd'hui, votre responsable vous demande de réaliser en autonomie un changement de format complet : passage du format Flacons + Gel douche (liquide) au format Pots + Granulés (solide).

Ce TP porte sur la préparation du changement de format, les réglages mécaniques et paramétriques, la qualification de la production et l'analyse SMED. Avant de commencer les questions, regardez la vidéo de rappel ci-dessous.

Répondez à chaque question puis cliquez sur « Valider ». Vous obtenez immédiatement un code couleur : vert = correct · rouge = à revoir.

Machine : POLYPROD PP30 – Ligne ERMAFLEX
Constructeur : ERM Automatismes Industriels – Carpentras (84)
Format départ : Flacons CRU 125 + Gel douche (liquide)
Format cible : Pots PV 125 + Granulés POLYSTYROL (solide)
Cadence maxi : 600 pots/h ou 600 flacons/h
Prérequis : Changement de format simple réalisé en 2nde Pro
